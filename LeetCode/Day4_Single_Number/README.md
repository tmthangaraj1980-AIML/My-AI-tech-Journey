# Day 4 - Single Number (LeetCode 136)

## Problem
Find the element that appears only once.

Example:
Input: [5,1,5,2,1]
Output: 2

## Approaches

### 1. Basic (count)
- Check frequency using count()

Time: O(n²)

### 2. Built-in (Math Trick)
- 2 * sum(set(nums)) - sum(nums)

Time: O(n)


## Learning Outcome
- Problem solving
- Multiple approaches
