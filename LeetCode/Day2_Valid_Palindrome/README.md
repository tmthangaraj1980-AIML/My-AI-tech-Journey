# Day 2 - Valid Palindrome (LeetCode #125)

## Problem
Check whether a string is a palindrome by ignoring spaces, symbols, and letter case.

## Approaches

### 1. Basic Approach
- Clean the string using isalnum()
- Convert to lowercase
- Reverse and compare
- Time Complexity: O(n)
- Space Complexity: O(n)

### 2. Two Pointer Approach
- Use left and right pointers
- Skip non-alphanumeric characters
- Compare characters
- Time Complexity: O(n)
- Space Complexity: O(1)

## Learnings
- String cleaning
- isalnum() usage
- lower() usage
- Two pointer technique
