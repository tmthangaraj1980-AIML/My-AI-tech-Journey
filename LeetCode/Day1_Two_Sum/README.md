# Day 1 - Two Sum

## Problem
Find two numbers in an array that add up to a target.

## Approaches

### 1. Brute Force
- Check all pairs
- Time Complexity: O(n²)

### 2. Optimized (HashMap)
- Store numbers in dictionary
- Time Complexity: O(n)

## Learnings
- HashMap usage
- Time vs Space trade-off
